# Public WCAG4PDF repository with desktop installers and issue tracker 

## Overview

[PDF4WCAG Accessibility Checker](https://pdf4wcag.com) helps quickly and easily validate the accessibility of  PDF documents according to WCAG, PDF/UA-1, PDF/UA-2 and Well-Tagged PDF requirements.

It is powered by the veraPDF validation architecture and is identical to [veraPDF](https://verapdf.org) in Machine verifiable checks of PDF/UA-1, PDF/UA-2 and Well-Tagged PDF profiles.

Additionally, it includes experimental AI-based heuristics to validate appropriate use of the PDF semantics.

## Desktop application

This repository contains latest installers of the Desktop version of PDF4WCAG. It serves as a wrapper for the web version, but runs locally with no external connections and no Internet acccess.

## Reporting issues

Feel free to report any issues agaist both web and desktop versions of PDF4WCAG. The development team will do the best to react and potentially fix reported issues.
